# OpenSource Compass

AI-powered platform helping developers discover, understand, and contribute to open source projects.

## Features

- GitHub OAuth Authentication
- Open Source Discovery
- Good First Issue Finder
- AI Issue Explanation
- Contribution Tracking
- Personalized Recommendations

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- MongoDB
- GitHub API
- Auth.js
- Groq AI
